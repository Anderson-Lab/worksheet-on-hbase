    Installation

    Installing Java

    1. sudo apt-get install python-software-properties
    2. sudo apt-get install software-properties-common
    3. sudo add-apt-repository ppa:webupd8team/java
    4. sudo apt-get update
    5. sudo apt-get install oracle-java7-installer

    Installing HBase

    1. wget http://apache.claz.org/hbase/stable/hbase-1.1.4-bin.tar.gz
    2. tar xfvz hbase-1.1.4-bin.tar.gz
    3. sudo mkdir /usr/lib/hbase
    4. sudo mv hbase-1.1.4/ /usr/lib/hbase
    5. We need to change the JAVA_HOME directory. Open the start script with "sudo nano /usr/lib/hbase/hbase-1.1.4/conf/hbase-env.sh". Then find the line that read "# export JAVA_HOME=/usr/java/jdk1.6.0/" and change it to "export JAVA_HOME=/usr/lib/jvm/java-7-oracle". Please note how I removed the # in addition to changing the directory.
    6. Now open your .bashrc file with "nano ~/.bashrc" and add the following line at the end "export HBASE_HOME=/usr/lib/hbase/hbase-1.1.4". Then save this file and exit nano.
    7. Now type "bash" and hit enter.

    Starting hbase

    1. sudo $HBASE_HOME/bin/start-hbase.sh
    2. sudo $HBASE_HOME/bin/hbase shell
